🎯 2단계: 설계 방침

ℹ️
기존 "P" 버튼 강제정지 / 재초기화 토글 로직(App_Bms_EnterForceStop / App_Bms_ReinitRuntime)을 그대로 재사용하여 코드 중복 없이 구현합니다. 센서가 Danger를 감지하면 내부적으로 "P가 눌린 것"과 동일한 경로를 타도록 합니다.
⚠️
자동 강제정지 이후 "P" 버튼 입력만이 유일한 해제 수단입니다. 센서가 정상으로 돌아와도 자동 해제되지 않습니다. 사람이 직접 확인하고 재가동해야 합니다.
요구사항 구현 방법 상태
센서 Danger → 모든 행위 정지 App_Bms_CheckDangerAutoStop() → App_Bms_EnterForceStop() 호출 완료
Manual / Auto 모드 구분 없이 정지 force_stop_lock=1 → ST_MACHINE() 실행 차단, Car_Stop() 완료
TraceInit (상태머신 초기화) App_Bms_EnterForceStop() 내부에 STMACHINE_Init() 추가 완료
Track / Charge CAN OFF 기존 App_Bms_EnterForceStop()에 이미 포함 (그대로 동작) 완료
정지 상태에서 P 외 모든 입력 무시 기존 App_Bms_NotifyRemoteCmd()의 force_stop_lock 체크 그대로 적용 완료
P 버튼으로만 해제 App_Bms_HandleForceToggle() → App_Bms_ReinitRuntime() 완료
자동/수동 정지 구분 배너 danger_auto_stopped 플래그 → 배너 "DANGER_AUTO_STOP" / "FORCE_STOP" 완료

③ App_Bms_CheckDangerAutoStop() — 신규 함수

/* =========================================================
 * 센서 Danger 자동 강제정지
 * ---------------------------------------------------------
 * 가스 / 전류 / 전압 / 온도 중 하나라도 Danger 상태가 되면
 * 수동으로 "P"를 누른 것과 동일하게 강제정지 처리한다.
 *
 * - force_stop_lock == 0 일 때만 진입
 * - force_stop_lock == 1 이면 이미 정지 중이므로 재진입 없음
 * - 해제는 반드시 "P" 버튼으로만 가능 (App_Bms_ReinitRuntime)
 * ========================================================= */
static void App_Bms_CheckDangerAutoStop(void)
{
    if (g_app_bms.force_stop_lock != 0U)
    {
        /* 이미 force stop 상태 → 추가 처리 불필요 */
        return;
    }

    if (BMS_SAFETY_IsDangerNow() == 0U)
    {
        /* 현재 danger 없음 → 정상 동작 */
        return;
    }

    /* Danger 감지 → 자동 강제정지 진입 */
    g_app_bms.danger_auto_stopped = 1U;

    printf("[APP] DANGER AUTO STOP reason=%s → force stop engaged\r\n",
           BMS_SAFETY_GetNowReason());

    App_Bms_EnterForceStop();
}

④ App_Bms_Task() — 4-1 단계 호출 추가

    /* 4) safety 정책 갱신 */
    BMS_SAFETY_Task();

    /* 4-1) Danger 자동 강제정지 체크
     * safety 갱신 직후 danger 여부를 확인하여
     * Danger 상태 진입 시 자동으로 force stop 처리한다.
     * (수동 P 버튼과 동일한 효과, P 버튼으로만 해제 가능)
     */
    App_Bms_CheckDangerAutoStop();

    /* 4-3) BMS 제한 재적용 */
    SafeDrive_ReapplyLimit();

⑤ App_Bms_GetBanner() — 자동/수동 구분 배너

    if (g_app_bms.force_stop_lock != 0U)
    {
        if (g_app_bms.danger_auto_stopped != 0U)
        {
            return "DANGER_AUTO_STOP";
        }
        return "FORCE_STOP";
    }

🌡️ 5단계: Danger 감지 센서별 조건

센서 Danger 진입 조건 Debounce 비고
🌡️ 온도 (NTC ADC) ADC 55°C 이상 (TEMP_STATE_DANGER) 또는 ADC 불량(TEMP_STATE_FAULT) 즉시 PC2 핀, ADC1 Rank 1
💨 가스 (ADC DMA) baseline 대비 delta ≥ 400 (GAS_DANGER) 또는 GAS_FAULT 800ms (2회) PC3 핀, ADC1 Rank 2, spike 필터 적용
⚡ 전압 (INA219) ≤ 5000mV (VOLT_UNDER_DANGER) 또는 ≥ 13500mV (VOLT_OVER_DANGER) 즉시 I2C1, electrical 그룹으로 묶임
🔌 전류 (INA219) ≥ 2500mA (CURRENT_STATE_DANGER) 즉시 I2C1, electrical 그룹으로 묶임
